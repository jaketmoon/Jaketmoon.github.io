c语言常用函数，占位符

发表于2024-01-11|更新于2024-01-18
|阅读量:

在考试的时候，有一个非常好用的函数可以达到事半功倍的效果，但是自己常常会忘记文件头以及对应函数怎么写。这里我将列出几个可供参考的函数

#include<stdio.h>

1.

#include<string.h>

有些对字符数组的操作只能使用这些函数

1.字符串复制函数strcpy

char *strcpy(char *s1,char *s2)

从c位置开始（包括 c）往后面加“water”，会覆盖原有数据，而且会在你输入的数据之后加“\0”!

结果是rain water

2.字符串连接函数strcat

char *strcat(char *s1,char *s2)

直接在s1的位置后面链接s2

结果是0123456789rain

3.字符串比较函数strcmp

int strcmp(char *s1,char *s2)

当结果==0，s1=s2

当结果>0, s1>s2

当结果<0, s2>s1

4.字符串长度函数strlen

unsigned int strlen(char *s1)

返回值为”\0”前的长度

#include<ctype.h>

1.字符检测函数isascii

int isascii(x)

x为字符：返回非0

x为非字符：返回0

2.数字检测函数isdigit

int isdigit (x)

x为数字：返回非0

x为非数字：返回0

3.英文字符检测函数isalpha

int isalpha(x)

x为英文字符：大写返回1,小写返回2

x为非英文字符：返回0
